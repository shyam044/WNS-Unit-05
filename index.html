It mainly focuses on three layers:

  • Physical Layer Security
    • In this layer, it stops outsiders from connecting to the wireless network.
    • Uses shielding materials to reduce the strength of the signal.
    • RF Coverage Area:
      • It ensures that the network signals cover only the right area.
  • Network Layer Security
    • In this layer, it protects the wireless network from attackers.
    • It uses protocols like WPA2 and WPA3.
    • It uses 802.1X authentication and VPN for remote users.
  • Application Layer Security
    • In this layer, it keeps the data safe and ensures that only authorized users can access the application.
    • It uses protocols like HTTPS, SFTP, and SSH.
    • It uses SSL/TLS and VPN to encrypt the data.
  • VPN Tunnelling:
    • The process of transmitting the data securely from a VPN Client to a VPN Server through a tunnel is called as VPN Tunnelling
  • Encryption:
    • The process of converting plaintext to ciphertext by using a cryptographic algorithm is called as Encryption.
  • Authentication:
    • The process of verifying the identity of a user or device is called as Authentication.
    • Only authorized users can access the data.

3) VPN Tunnelling

  • The process of transmitting the data securely from a VPN Client to a VPN Server through a tunnel is called as VPN Tunnelling
  • Uses of VPN Tunnelling:
    • It provides Security
    • It provides secure communication
    • It provides Encryption
    • It prevents attacks from Man-In-The-Middle attacks, DOS attacks and eavesdropping

5) Retail and Manufacturing Design

Roaming Agent and Kiosk devices are a wireless network architecture used in Retail and Manufacturing Design

  • Servers:
    • They provide services like DNS, DHCP, Logging and Applications
  • Firewall:
    • Protects the Network from unwanted traffic
  • VPN Gateway:
    • It provides secure communication between access points and the server
  • Access Points:
    • It acts as a bridge between wired and wireless networks
  • Roaming Agents:
    • Roaming Agents are PDA with Scanners
  • Kiosk devices:
    • Kiosk devices are PDA with Printers

    • Advantages:

    • They provide security
    • They provide secure communication
    • Simplicity
    • Mobility

    Disadvantages:

    • Uses Old Security models like WEP
    • It requires backend servers; failure in that server affects the entire system
    • Not easy to maintain

8) Intrusion Detection Systems (IDS)

  • IDS stands for Intrusion Detection Systems
  • IDS is a Security tool used to detect and react to security attacks
  • Types of IDS:
    • There are mainly two types of IDS. They are:
      1. HIDS
      2. NIDS
  1. HIDS:
    • HIDS stands for Host-Based IDS
    • It is installed on individual computers
    • HIDS may reduce the system Performance
  2. NIDS:
    • NIDS stands for Network-Based IDS
    • It monitors the Network traffic
    • NIDS is used for enterprise networks
  • Servers:
    • They provide services like DNS, DHCP, Logging and Applications
  • Firewall:
    • Protects the Network from unwanted traffic
  • VPN Gateway:
    • It provides secure communication between access points and the server
  • Access Points:
    • It acts as a bridge between wired and wireless networks
  • PDA/Device:
    • PDA stands for Personal Digital Assistant

    • Note:

    • IDS do not give a guarantee to prevent all types of attacks
    • But it gives a guarantee to detect all types of attacks
